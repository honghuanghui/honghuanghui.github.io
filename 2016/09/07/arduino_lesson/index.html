使用树莓派、arduino和esp8266通过微信客户端实现物联网的完整教程（实践篇、含源码）

##导言

这篇实践篇的教程是以上一篇原理篇的教程为依据，从客户端到服务器端，再到下位机这三个大方面进行实际操作的讲解。帮助读者在阅读完本教程之后，能够亲自搭建出属于自己的物联网平台。当然在此之前，你应该具备一些基本的知识储备，如有遇到不懂的地方，欢迎大家互相交流，共同进步。

##主要器材：
树莓派、arduino、esp8266、TTL转串口线、面包板（选用）、手机、电脑、红外发射二极管、红外接收二极管、温度传感器。

##树莓派搭建服务器：

###一、基础工作

####1.树莓派基本设置
1.安装树莓派的linux发行版系统，当然，也可以选择其他适合树莓派的系统。
如图：

2.在路由器的控制界面做端口映射，然后将树莓派联入互联网。客户端通过预先映射的端口访问路由时，路由器就会将其映射到你的树莓派上。所以你也可以把路由看做是树莓派与外网连接的一座桥梁，这样就可以远程访问树莓派了。我分别将22、80端口映射到树莓派，如图：

图中的IP地址就是树莓派的IP地址。

3.使用ssh通讯协议，远程/局域网控制树莓派，对其进行操作。上面展示的通过终端连接树莓派的图就是通过SSH进行连接的。在树莓派上安装SSH的服务器软件，然后用电脑上的客户端连接即可。你也可以在你的手机上下载一个ssh的客户端软件，这样你就可以随时随地对你家的树莓派进行操作了。

4.遇到不懂的Linu命令或者相关的问题，完全可以Google一下，很多大牛已经为你解答了！

###二、web服务器搭建

####1.安装 Apache

Apache 可以用下面的命令来安装
sudo apt-get install

Apache 默认路径是
/var/www/

其配置文件路径为
/etc/apache2/

可以通过sudo vi /etc/apache2/ports.conf修改监听端口号

重启服务生效
sudo service apache2 restart

####2.安装 mysql

sudo apt-get install mysql-server 安装过程中，会出现一个提示符让你输入一个密码。 这个密码是 mysql root 用户的密码。

####3.安装 PHP

输入下面的命令，就可以安装 PHP 5,以及 PHP 访问 mysql 数据库所需要的库。

sudo apt-get install php5
sudo apt-get install php5-mysql

####4.测试

安装完成后，可以在浏览器中输入你路由器的 IP 或域名，就可以访问你的网站了。你应该能看到一个页面显示“It works”，但是没有其它内容。

###三、原理解析：
esp8266与arduino的通信的通过串口实现的，借助着arduino强大的第三方库，我们在它上面定义两个IO引脚作为TTL串口的引脚（分别为7、8引脚），用软件去实现串口的功能。

esp8266:
它是一块WIFI模块的板子，其有三种模式，而我们使用的是客户端模式。开启此模式，就可以像电脑一样连接你们家的无线路由器了。同时，esp8266有很多种型号，我用的是esp8266-01，它有两个GPIO口，所以如果你是想做个小实验，控制一个LED灯的话，那么完全可以不需要arduino，用这块板子就可以实现了。另外，值得关注的是，esp8266的高版本引出了更多的GPIO，而且功能更为强大。你完全可以靠它去实现多种功能的控制。

Arduino:这块板子大家应该很熟悉了吧，我这里就不做介绍了。需要说明一下的就是，我们用到了温度传感器和红外控制的库，所以需要下载相关的库放入library中。这里提供几位大神用arduino控制空调和使用温度传感器的资料。

##手机微信客户端：

###一、基础知识

因为我们是在一个公众号里面发送指令的，所以我们需要去申请一个微信公众号。本着这篇文章的宗旨，这里我再插一段：我们需要一个访问树莓派的外网IP，也就是你的路由器外网IP。如果你家里的路由器使用的是动态IP，外网IP是会改变的。不过！这难不倒我们，我们可以用一个域名去代替动态IP，这样不管外网IP怎么变，我们只要访问这个域名就可以访问到我们的树莓派了。是不是很方便呢！

###二、公众号设置

1.进入公众号的设置页面中。

2.在URL中写入你的域名，并且token要与后端的程序中设置的相同，否则会报错。如果设置成功，那么表示这个公众号已经成功访问到了你家的树莓派了！

这时，你需要用手机关注这个公众号，此时你应该看到有公众号回复的信息，回复的内容为：欢迎来到智慧家居！
如果没有回复，说明服务器端的程序有问题。你需要检查你的后端程序，并且确保公众号与服务器建立了连接。

##检查并调试

###一、检查
如果以上的工作已经完成并且确保各个环节都没有出错的情况下，那么恭喜你，你即将可以远程控制你想要控制的设备了。此时，你应该再次检查你的下位机接线，并且确保树莓派已经联网。

###二、调试
拿起你的手机，关注了你刚刚创建的公众号之后，输入内容为：温度。此时你应该会看到公众号回复一个包含温度的信息，如果收到了，那么你已经成功了！接下来就是控制空调等等你希望控制的设备了！

如果没有收到，你就要仔细检查各个环节是否有出错的地方了。记得我当时前几次都没有成功，后来检查到是公众号设置有错误。
